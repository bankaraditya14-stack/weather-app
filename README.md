# Weather App 🌤️

A compact, responsive weather application that fetches real-time weather data for any city using the OpenWeatherMap API.

## Features

- 🔍 **Search by City**: Enter any city name to get current weather
- 🌡️ **Temperature Display**: Shows current temperature, feels-like temperature, and weather description
- 💧 **Detailed Info**: Displays humidity, wind speed, and atmospheric pressure
- 📱 **Responsive Design**: Works seamlessly on desktop, tablet, and mobile devices
- ⚡ **Real-time Data**: Fetches live weather information from OpenWeatherMap API
- 🎨 **Modern UI**: Clean, gradient design with smooth animations
- ⌨️ **Keyboard Support**: Press Enter to search

## How to Use

1. Open `index.html` in your web browser
2. Enter a city name in the search box
3. Click "Search" or press Enter
4. View the current weather information for that city

## API Used

- **OpenWeatherMap API** - Free tier provides current weather data
- API Key is included for demo purposes
- For production use, consider using your own API key from [openweathermap.org](https://openweathermap.org/api)
